# 05. 저작권 및 법적 리스크 관리 정책

## 1. 기본 원칙

Daily Brief Note의 모든 콘텐츠는 공개 자료를 참고하되, 표현과 구성은 독립적으로 재작성한다.

사실 정보는 참고할 수 있지만, 기사나 자료의 고유한 문장 표현을 복제하지 않는다.

---

## 2. 금지되는 행위

### 기사와 문서

- 기사 전문 복사
- 기사 문단 대량 복사
- 제목과 리드를 그대로 재사용
- 해외 기사 전체 직역
- 여러 기사의 문장을 이어 붙여 새 글처럼 구성
- 유료 기사 내용 재현
- 논문·보고서의 표와 설명을 무단 복제

### 방송과 영상

- 방송 자막 대량 복제
- 영상 내용을 전체 번역
- 방송 화면 캡처를 대표 이미지로 사용
- CCTV 원문 전체를 학습 콘텐츠에 재현

### 이미지

- 언론사 사진 무단 사용
- 검색 결과 이미지를 출처 확인 없이 사용
- 기업 공식 이미지를 이용 조건 확인 없이 사용
- 타인의 인포그래픽·도표 복제
- 워터마크 제거 후 사용

---

## 3. 허용되는 작성 방식

다음 순서로 독립적으로 재구성한다.

1. 무엇이 확인되었는가
2. 왜 중요한가
3. 독자에게 어떤 영향을 미치는가
4. 앞으로 무엇을 확인해야 하는가

단순히 단어만 바꾸는 방식은 재작성으로 인정하지 않는다.

---

## 4. 인용 원칙

- 인용은 꼭 필요한 최소 범위만 사용
- 인용문과 작성자의 설명을 명확히 구분
- 인용 출처를 바로 표시
- 인용문이 글의 대부분을 차지하지 않게 함
- 원문 의미를 왜곡하지 않음

---

## 5. 중국어 학습 콘텐츠 특별 규칙

### 허용

- 학습에 필요한 핵심 문장 3~5개
- 짧은 표현이나 구문
- 병음·문장 구조·단어 설명
- 독립적으로 만든 응용 문장
- 원문 링크 제공

### 금지

- CCTV 기사 전체 복사
- 전체 자막 복사
- 기사 전체 번역
- 학습 문장을 임의로 만들어 원문이라고 표시
- 원문에 없는 문장을 CCTV 표현으로 소개
- 개별 원문 URL 없이 공식 홈페이지 주소만 제공

---

## 6. 출처 기록

출처에는 가능한 범위에서 다음을 포함한다.

- 기관 또는 언론사명
- 원문 제목
- 게시일 또는 업데이트일
- 개별 원문 URL
- 해당 자료에서 확인한 핵심 내용

---

## 7. 이미지 저작권

가장 안전한 순서:

1. 직접 생성한 이미지
2. 직접 촬영한 이미지
3. 공식적으로 재사용이 허용된 이미지
4. 라이선스가 명확한 스톡 이미지

언론사 사진과 방송 화면은 사용하지 않는다.

---

## 8. 사실과 의견 구분

다음 표현을 구분한다.

- 공식 발표에 따르면
- 보도에 따르면
- 업계에서는
- 전문가들은
- 이 글에서는 다음과 같이 해석한다

의견이나 전망을 사실처럼 작성하지 않는다.

---

## 9. 발행 전 저작권 체크리스트

- 기사 문장을 그대로 복사하지 않았는가
- 해외 기사를 직역하지 않았는가
- 원문보다 과도하게 상세한 재현이 없는가
- 인용은 최소 범위인가
- 출처가 정확히 표시되었는가
- 언론사 사진을 사용하지 않았는가
- 표·도표·인포그래픽을 무단 복제하지 않았는가
- CCTV 학습 문장이 실제 원문에 존재하는가
- 원문 전체가 아닌 핵심 문장만 사용했는가
- 대표 이미지의 사용 권한이 명확한가
