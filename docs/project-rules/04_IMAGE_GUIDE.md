# 04. 대표 이미지 제작 가이드

## 1. 기본 사양

- 비율: 16:9
- 권장 크기: 1200×675px
- 최종 형식: WebP
- 권장 용량: 300KB 이하
- 텍스트 없음
- 로고 없음
- 워터마크 없음
- 기사 제목 삽입 금지

사용자가 `webP형식으로 압축`이라고 요청하면 다음 조건을 기본 적용한다.

- WebP 형식
- 1200×675px
- 300KB 이하 권장

---

## 2. 공통 이미지 원칙

- 현실적이고 전문적인 미디어 스타일
- 과도한 공포·충격 표현 금지
- 피해 장면 과장 금지
- 실제 사건과 다른 장면 생성 금지
- 특정 언론사 사진이나 방송 화면 모방 금지
- 인물의 신원이나 행동을 오해하게 만드는 구성 금지
- 실제 로고·상표·뉴스 자막을 삽입하지 않음
- 텍스트 없는 인터페이스와 상징 요소 사용 가능

---

## 3. 카테고리별 스타일

## 3.1 뉴스 브리핑

### 권장 스타일

- 현실적인 전문 뉴스·산업 미디어
- 신뢰감 있는 색감
- 차분한 청회색, 중립색, 필요한 경우 경고성 주황색
- 사건의 핵심 요소를 시각적으로 연결
- 과도한 재난·공포 표현 금지

### 프롬프트 구조

`현실적인 전문 [분야] 뉴스 미디어 스타일의 16:9 대표 이미지. [핵심 장면]. [보조 요소]. 차분하고 신뢰감 있는 색감, 과도한 공포 표현 없음, 텍스트 없음.`

---

## 3.2 AI 칼럼

### 권장 스타일

- 현실적인 전문 IT·산업 미디어
- 현대적인 업무 환경
- AI 시스템, 데이터 흐름, 인간 감독의 균형
- 밝고 신뢰감 있는 색감
- 추상적인 로봇 얼굴보다 실제 활용 환경 우선

### 프롬프트 구조

`현실적인 전문 IT·산업 미디어 스타일의 16:9 대표 이미지. [AI 개념을 보여주는 중심 장면]. [데이터·업무·도구 연결 요소]. 인간 감독과 기술 활용의 균형, 깔끔하고 신뢰감 있는 분위기, 텍스트 없음.`

---

## 3.3 정보DB

### 권장 스타일

- 교육적 인포그래픽
- 전문 지식 미디어
- 개념 구조가 한눈에 보이도록 구성
- 너무 많은 요소를 넣지 않음
- 배경과 핵심 요소의 대비 확보

### 프롬프트 구조

`전문 교육·지식 미디어 스타일의 16:9 대표 이미지. [핵심 개념]을 중심에 배치하고 [구성 요소 또는 비교 요소]를 시각적으로 연결한다. 정돈된 인포그래픽 분위기, 높은 가독성, 텍스트 없음.`

---

## 3.4 중국어 학습

### 권장 스타일

- 중국 뉴스와 언어 학습 요소 결합
- CCTV 화면을 직접 복제하지 않음
- 중국어 문자를 이미지에 직접 삽입하지 않음
- 뉴스 스튜디오, 모바일 학습, 병음·문장 구조를 상징하는 비문자 인터페이스 활용
- 교육적이고 전문적인 분위기

### 프롬프트 구조

`현실적인 전문 중국어 교육 미디어 스타일의 16:9 대표 이미지. [뉴스 주제]를 배경으로 중국어 학습과 문장 분석을 상징하는 텍스트 없는 인터페이스 요소를 배치한다. 깔끔하고 신뢰감 있는 교육 분위기, 텍스트 없음.`

---

## 4. 이미지 ALT 문구

ALT 문구는 이미지 내용을 객관적으로 설명한다.

### 기준

- 15~40자 권장
- 핵심 주제 포함
- `이미지`, `사진` 같은 불필요한 단어 생략
- 키워드 반복 금지
- 이미지에 없는 내용 삽입 금지

### 예시

- AI 에이전트가 업무를 단계별로 수행하는 디지털 작업 환경
- 반도체와 데이터센터가 연결된 산업 네트워크
- CCTV 뉴스 원문을 활용한 중국어 학습 장면
- 지진 관측 정보와 긴급 대응 현장이 결합된 재난 뉴스 장면

---

## 5. 금지 사항

- 언론사 사진 무단 사용
- 방송 화면 캡처 사용
- 실제 인물 얼굴을 허위 상황에 배치
- 사망·부상 장면의 과도한 묘사
- 공포·혐오·충격 중심의 연출
- 이미지 안에 제목·설명·로고 삽입
- 기사 내용과 관계없는 장식 이미지
- 해상도만 높고 정보 전달이 불명확한 이미지
- 16:9 비율 미준수
- WebP 압축 요청 시 PNG/JPEG로만 제공
