# 03. SEO 최종 규칙

## 1. 출력 순서

각 글은 다음 순서로 출력한다.

1. SEO 입력용 대표 제목
2. SEO 대안 제목 4개
3. 메타 설명
4. URL 슬러그
5. 포커스 키워드
6. SEO 태그
7. 워드프레스 본문용 HTML
8. 대표 이미지 프롬프트
9. 이미지 ALT 문구
10. 발행 전 체크리스트

SEO 정보 전체를 하나의 코드블록으로 묶지 않는다.

---

## 2. SEO 제목

### 공통 기준

- 대표 제목 1개
- 대안 제목 4개
- 핵심 키워드를 자연스럽게 포함
- 제목만 보고 글의 내용을 이해할 수 있게 작성
- 과도한 클릭 유도 금지
- 확정되지 않은 사실 단정 금지
- 본문에 없는 내용 삽입 금지

### AI 칼럼

`[글 제목] - AI-###`

### 정보DB

`[글 제목] - 정보DB-###`

### 중국어 학습

`CCTV 뉴스로 배우는 중국어 #[번호]｜[뉴스 주제] 핵심 표현 정리`

### 뉴스 브리핑

`[핵심 이슈 1], [핵심 이슈 2] - [날짜] [카테고리] 브리핑｜`

---

## 3. 메타 설명

- 권장 길이: 120~160자
- 핵심 주제를 앞부분에 배치
- 같은 키워드를 반복하지 않음
- 제목을 그대로 복제하지 않음
- 실제 글에서 제공하는 내용만 설명
- 과장·확정·선정적 문구 금지

---

## 4. URL 슬러그

### 공통 규칙

- 영어 소문자
- 숫자 사용 가능
- 단어 구분은 하이픈
- 한글 금지
- 공백 금지
- 특수문자 금지
- 지나치게 긴 주제형 슬러그 금지

### 뉴스 브리핑

- 종합: `daily-news-briefing-YYYY-MM-DD`
- 경제: `economy-briefing-YYYY-MM-DD`
- 국제: `global-briefing-YYYY-MM-DD`
- 과학기술: `technology-briefing-YYYY-MM-DD`
- 사회: `society-briefing-YYYY-MM-DD`
- 환경·에너지: `climate-energy-briefing-YYYY-MM-DD`

### AI 칼럼

`ai-###`

### 정보DB

`info-db-###`

### 중국어 학습

`cctv-chinese-news-###`

`###`는 최소 3자리로 zero-padding하며 1000 이상은 자릿수를 제한하지 않는다. 이미 발행된 게시물의 slug와 WordPress URL은 규칙 변경만으로 자동 수정하지 않는다.

---

## 5. 포커스 키워드

포커스 키워드는 글의 핵심 검색 의도를 하나의 자연스러운 구문으로 작성한다.

예시:

- 생성형 AI란 무엇인가
- 피지컬 AI
- 큐비트란 무엇인가
- CCTV 뉴스 중국어
- 오늘 경제 뉴스 브리핑

키워드를 본문에 억지로 반복하지 않는다.

---

## 6. SEO 태그

태그는 **5~8개만 출력**한다.

### 포함 기준

- 본문 핵심 주제와 직접 관련
- 독자가 실제 검색할 가능성이 있음
- 서로 의미가 겹치지 않음
- 지나치게 넓거나 모호하지 않음

### 제외 기준

- 카테고리명
- `Daily Brief Note`
- `DailyBriefNote`
- 의미가 중복되는 태그
- 단수·복수만 다른 유사 태그
- 본문과 관련 없는 인기 키워드
- 제목 전체를 그대로 복사한 태그

### 잘못된 예

- AI, 인공지능, 인공지능기술, AI기술
- 중국어 학습, 중국어공부, 중국어배우기
- 뉴스 브리핑, 경제 브리핑, 경제 뉴스 브리핑

유사 의미는 하나만 선택한다.

---

## 7. 내부 링크

각 글에는 관련성이 높은 이전 콘텐츠 1개 이상을 연결한다.

- 뉴스: 이전 관련 브리핑
- AI 칼럼: 이전 AI 칼럼
- 정보DB: 이전 정보DB
- 중국어 학습: 이전 학습 콘텐츠

관련성이 없는 글을 억지로 연결하지 않는다.

---

## 8. FAQ

- 권장 문항 수: 3~5개
- 실제 검색 의도와 연관된 질문 사용
- 본문을 그대로 반복하지 않음
- 짧고 명확하게 답변
- 새로운 핵심 정보를 추가할 수 있음
- 과장된 답변 금지

---

## 9. 발행 전 SEO 체크리스트

- 대표 제목에 핵심 키워드가 자연스럽게 포함되었는가
- 대안 제목 4개가 서로 다른 검색 의도를 반영하는가
- 메타 설명이 120~160자인가
- URL 슬러그가 형식에 맞는가
- 포커스 키워드가 하나의 명확한 구문인가
- 태그가 5~8개인가
- 카테고리명이 태그에서 제외되었는가
- 중복 태그가 제거되었는가
- 내부 링크가 관련성이 있는가
- 본문에 없는 내용이 제목·메타 설명에 들어가지 않았는가
